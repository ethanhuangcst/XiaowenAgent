# 我要做什么

## 第一步：整理需求 ✅

### 1. 这个Agent要解决什么问题？
这个Agent叫做 **XiaowenAgent**，用来帮我优化微信公众号文章和视频号图文。

### 2. 用户是谁？使用场景是什么？
**用户**：我自己

**使用场景**：
- 优化大纲
- 生成内容
- 定时发布

### 3. 核心功能有哪些？

**功能一：微信公众号涨粉专家 - 优化大纲**
- 输入：Markdown格式的大纲（单篇或系列，系列居多）
- 输入：优化要求
- 输出：优化后markdown格式的大纲（单篇或系列，系列居多）

**功能二：逐篇产生内容**
针对每一篇内容，生成配套的公众号文章+视频号图文内容草稿：
- 使用微信内容达人的技能
- 参考我提供的范文，学习、了解我的文风
- 每一篇内容包含：
  - 一篇公众号文章（500字以内，内容是全文）
  - 一篇视频号图文（不算标题，6-7个带标题文字的图片，是这一篇内容的概要）
- 格式：Markdown，方便编辑后反向输入回给智能体，然后定稿
- 流程：一篇内容定稿之后，再生成下一篇

**功能三：定时发布**
- 按照预设的时间，定时发公众号文章
- 定时发布视频号图文
- 公众号文章和视频号图文发布成功后，在我的微信朋友圈发两条转发

### 4. 成功的标准是什么？
- 视频图文view 1000+
- 公众号文章view 1000+
- 自动发布的三个路径都打通

### 4. Agent如何与用户交互？
**方案A - Web界面**

## 第三步：实现Agent核心 ✅

### 1. Agent状态管理
**状态机设计**：
```
IDLE（空闲）→ PLANNING（规划）→ EXECUTING（执行）→ CONFIRMING（确认）→ IDLE
```

**状态说明**：
- **IDLE**：等待用户输入
- **PLANNING**：分析任务，制定计划
- **EXECUTING**：执行具体任务（调用LLM、数据库等）
- **CONFIRMING**：等待用户确认

### 2. Agent决策逻辑
**固定流程决策**：
```
步骤1：优化大纲
  - 调用LLM优化大纲
  - 返回优化后的大纲
  - 进入CONFIRMING状态

步骤2：用户确认大纲
  - 用户编辑/确认大纲
  - 如果确认，进入下一步
  - 如果拒绝，返回步骤1

步骤3：逐篇生成内容
  - 针对每一篇生成内容
  - 生成公众号文章（500字以内）
  - 生成视频号图文（6-7张图片）
  - 进入CONFIRMING状态

步骤4：用户确认内容
  - 用户编辑/确认内容
  - 如果确认，定稿
  - 如果拒绝，返回步骤3

步骤5：定时发布
  - 按预设时间发布
  - 公众号文章
  - 视频号图文
  - 朋友圈转发
```

## 第四步：实现具体功能 ✅

### 1. 大纲生成功能
**输入**：
- Markdown格式的大纲（单篇或系列）
- 优化要求（可选）

**处理流程**：
```
1. 解析Markdown大纲
2. 提取关键信息（标题、Key Message、状态等）
3. 调用LLM优化大纲
   - 补充缺失字段（钩子、标签、关键词）
   - 优化标题吸引力
   - 确保逻辑连贯
4. 返回优化后的Markdown大纲
```

**输出**：
- 优化后的Markdown大纲
- 用户可编辑
- 支持重新生成

### 2. 内容生成功能
**输入**：
- 完整大纲（已生成并确认）
- 用户偏好（从记忆系统获取）

**处理流程**：
```
循环处理每一篇：
1. 自动获取下一篇大纲
2. 生成公众号文章
   - 字数：500字以内
   - 内容：全文
   - 格式：Markdown
   - 参考：用户范文风格

3. 生成视频号图文
   - 数量：6-7张图片
   - 每张：带标题的文字图片
   - 内容：文章概要
   - 格式：Markdown

4. 返回内容草稿
5. 等待作者定稿
6. 定稿后，进入下一篇
7. 重复步骤1-6，直到所有篇完成
```

**输出**：
- 公众号文章（Markdown）
- 视频号图文（Markdown）
- 用户可编辑
- 支持重新生成
- 定稿后自动进入下一篇

### 3. 图片生成功能
**输入**：
- 视频号图文内容（Markdown）
- 图片风格要求

**处理流程**：
```
1. 解析图文内容
2. 为每张图片生成提示词
3. 调用图片生成API
   - 通义万相（默认）
   - 或其他图片生成服务
4. 返回图片URL列表
```

**输出**：
- 图片URL列表
- 图片预览
- 支持重新生成

### 4. 发布功能
**输入**：
- 定稿内容
- 发布时间
- 发布平台

**处理流程**：
```
1. 定时任务调度
2. 发布公众号文章
   - 调用微信公众号API
   - 上传内容
   - 发布

3. 发布视频号图文
   - 调用视频号API
   - 上传图片
   - 发布

4. 朋友圈转发
   - 发布成功后
   - 自动转发到朋友圈
   - 生成转发文案

5. 返回发布状态
```

**输出**：
- 发布状态（成功/失败）
- 发布链接
- 错误信息（如果失败）
